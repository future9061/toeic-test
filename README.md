## 📌 Convention & Rule

- _작업 전 이슈 생성_ : `ISSUE TEMPLATE`
- _브랜치 이름은 feat/해당 이슈 번호_
  - ex) feat/#1
- _feature 브랜치로 전환한 후 작업 진행하기_
- _작업 끝나면 add, commit, push_
  - _commit 시_ `COMMIT TEMPLATE`
- _feat 브랜치는 local 에서만 삭제, remote에서는 남겨두기_
